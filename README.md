# STS2 Mods

A collection of quality-of-life and gameplay-enhancing mods for Slay the Spire 2.

---

## Mods

### StartWithOrobas

Start each run with the Orobas relic '[Touch of Orobas](https://slaythespire.wiki.gg/wiki/Slay_the_Spire_2:Touch_of_Orobas)' automatically granted at the beginning of the game.

**Features:**
- Injects Orobas into the player’s starting relic pool
- Incorperates the mod [Mod Config](https://www.nexusmods.com/slaythespire2/mods/27) for easy disabling of mod without neededing to restart game


**Use Case:**
- Ideal for players who want a bit of an easier start

---

### RandomStart

Randomize each session with a new starting deck.

**Features:**
- Randomizes starting deck 
- Incorperates the mod [Mod Config](https://www.nexusmods.com/slaythespire2/mods/27) which allows the player to customize their start
- Allows the choice of how your starting deck is randomized


**Use Case:**
- Ideal for players who want variety without losing seed consistency

---

### StartUpgraded

Automatically upgrades your starting deck at the beginning of each run.

**Features:**
- Upgrades all starter cards immediately on run start
- Incorperates the mod [Mod Config](https://www.nexusmods.com/slaythespire2/mods/27)
- Allows the choice of how your starting deck is upgraded

**Design Goal:**
- Accelerate early-game pacing without altering core mechanics

---


### Deploy

Each mod requires the following files to be placed in:

<Slay the Spire 2>/mods/'MODNAME'/

| File | Purpose |
|------|--------|
| ModName.dll | Compiled mod |
| ModName.json | Mod loader manifest |
| ModName.pck | Resource pack (required for compatibility) |
